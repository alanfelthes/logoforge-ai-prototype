Open hosted checkout

const popup = window.open(
  response.checkout_url,
  "LogoForgeCheckout",
  "width=560,height=820"
);

window.addEventListener("message", (event) => {
  if (event.origin !== "https://pay.example.com") return;
  if (event.data.type !== "LOGOFORGE_PAYMENT_RESULT") return;

  // UI only. Confirm on the server.
  console.log(event.data.status);
});
Important: the popup callback is not payment proof. Fulfill only after the signed server webhook or payment-status API confirms the final state.

Required signature headers

LogoForge-Event-Id: evt_platform_01JZ
LogoForge-Timestamp: 1783347509
LogoForge-Signature: v1=<HMAC_SHA256>
Verification: compute HMAC over timestamp + "." + raw body using the website webhook secret.
